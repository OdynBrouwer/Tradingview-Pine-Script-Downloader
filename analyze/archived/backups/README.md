# backups/

Bevat een kleine set historische `.pine` backup-bestanden.

Aanbeveling:
- Controleer de bestanden (`*.pine`) en verwijder ze als je ze niet meer nodig hebt.
- Optioneel: verplaats permanent bruikbare backups naar een centrale backup-locatie (NAS/backup-folder).

Huidige inhoud (voorbeeld):
- `WOP_owner.pine`
- `zig_zag_indicator_ower.pine`
- `zig_zag_owner.pine`

Wil je dat ik deze bestanden automatisch verwijder of verplaats naar `analyze/archived/`? Geef 'delete' of 'move'.
